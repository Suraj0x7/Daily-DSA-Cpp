# 🚀 DSA with C++

This repository contains daily C++ solutions for Data Structures and Algorithms problems.
It is organized by problem sets and includes practice code for CampusMonk challenges.

## 📌 What’s inside

- `CampusMonk/` — C++ source files for CampusMonk practice problems
- `output/` — compiled executables and generated files (ignored by git)

## 🧠 Sample solutions included

- `1_count_all_digit.cpp`
- `2_count_the number_of_odd_digit.cpp`
- `7_Palindrome_number.cpp`
- `8_Largest_digit_in_number.cpp`
- `9_Divisors_of_a_number.cpp`
- `10_factorial_of_number.cpp`

## 🛠 Language

- C++

## 🚀 Usage

To compile and run a source file from the repository root:

```bash
cd CampusMonk
g++ -Wall -Wextra -g3 <filename>.cpp -o output/<filename>.exe
./output/<filename>.exe
```

Example:

```bash
g++ -Wall -Wextra -g3 10_factorial_of_number.cpp -o output/10_factorial_of_number.exe
./output/10_factorial_of_number.exe
```

## 📌 Notes

- The `output/` directory is used for compiled binaries and should not be committed.
- Keep source files in `CampusMonk/` and push only `.cpp` solutions.
